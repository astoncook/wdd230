My Website Name

The Drop

The website name "The Drop" reminds me of rain and also some type of alert. This is an atractive name for me because it can be percieved as multiple things and have a lot of meaning to it as a weather website.

Site Purpose

Informing

My site will inform users of weather statments for the present and future. They will be able to see the different forecasts for the week and they will be notified of speacial weather statments if any do occur. Some examples of that are severe thunderstorms, heat advisory, or tornado warnings.
